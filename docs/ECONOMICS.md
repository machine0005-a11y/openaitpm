# Economics

## Gate 0 Status

Stub only. The variable cost model is deferred until after Gate E when the real core loop can be measured.

## Cost Categories To Model

- LLM planner calls per MusicSpec generation.
- Music generation provider cost per rendered minute.
- Storage cost per audio asset and story card image.
- Egress cost for playback and story card downloads.
- Queue/worker runtime cost.
- Database cost for users, moments, summaries, events, and assets.

## Draft Pricing Axes

- Free: limited private moments and stub/demo generations.
- Creator: higher monthly generation quota and public share cards.
- Pro: larger audio quota, faster queue, export/deep-link modules.
